# Reflecto Homebrew Tap

Phone notifications in your terminal.

## Install

```
brew tap reflectoapp/cli
brew install reflecto
```

## Usage

```
reflecto pair                              # generates a 6-digit code
# Enter the code on your phone (Reflecto app → "Add a device")
reflecto tail                              # stream notifications
```

See https://reflecto.app for full docs.
